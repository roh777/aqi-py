## Air Quality Index Info in Terminal

Get AQI Index by city, in you terminal using https://aqicn.org API.

Note: Requires python 3

#### Usage

```
$ python aqi.py -c Seoul
```
or 

```
$ python aqi.py --city Seoul
```
